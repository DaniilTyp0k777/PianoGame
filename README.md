
# Piano Game

Небольшая игра/симулятор пианино на **Python + Pygame**. Можно играть с клавиатуры или мышкой, а также менять громкость и количество активных клавиш в меню настроек.

При запуске программа пытается открыть GitHub-страницу автора в браузере.

## Возможности

- **Игра на пианино** с реальными звуками (MP3).
- **Управление**: клавиатура и мышь.
- **Визуальная клавиатура** с подсветкой нажатых клавиш.
- **Настройки**:
  - громкость
  - количество активных клавиш

## Требования

- Python 3.9+ (рекомендуется)
- Библиотека `pygame`

## Установка

1. Установите зависимости:

```bash
pip install pygame
```

2. Убедитесь, что в проекте есть папка со звуками:

`assets/sounds/*.mp3`

## Запуск

Из папки проекта:

```bash
python main.py
```

## Управление

- **Клавиатура**: `A`, `B`, `C`, `D`, `E`, `F`, `G`, `Q`, `W`, `S`, `X`
- **Мышь**: клик по клавишам на экране
- **Настройки**: кнопка (иконка) в левом верхнем углу

## Структура проекта

- `main.py` — основной цикл игры, обработка событий, отрисовка
- `settings.py` — размеры окна, цвета, список клавиш и файлов звуков (`KEYS`)
- `sounds.py` — загрузка звуков из `assets/sounds/`
- `keys.py` — создание прямоугольников клавиш и отрисовка клавиатуры
- `buttons.py` — кнопки UI
- `assets/` — ресурсы (звуки/картинки)

## Примечание про автозапуск браузера

В начале `main.py` есть код, который открывает ссылку GitHub через `webbrowser.open(...)`. Если это мешает, можно удалить этот блок.
